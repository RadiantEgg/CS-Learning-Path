##  新学习
- 指针的const修饰
    - const int *p = int const *p : *p不能改
    - int *const p : p不能改
    - const int *const p : 指针和值都不能改
-  循环写法：
    -  ```c 
        for (i = 0; i < n; i++)
        printf("%d\n", a[i]);       
        ```
    - 换成：
    ```c
        for (p = a; p < a + n; p++)
            printf("%d\n", *p);
    ```
