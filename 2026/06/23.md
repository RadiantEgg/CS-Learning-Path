##  新学习
- errno 全局整数，记录最近一次系统调用错误码
- strerr(errno) 将错误码转换为文字字符串
- void perror(const char *str) 输出格式为：自定义str：errno的错误信息
    ===== fprintf(stderr, "%s:%s", str, strerr(errno))
- int feof(FILE *stream) 文件读写位置指针越过文件末尾返回非0值
- int ferror(FILE *stream) 上一次读写出错返回非0值
    - 在fopen后ferror自动设置为0
- void clearerr(FILE *stream) 清楚文件错误标志ferror()和文件结束标志feof()
    - 只要出现读写错误，错误标志ferror会一直存在，直到调用clearerr

```c
#include <stdio.h>
#include <stdlib.h>
#include <stderr.h>

int main()
{
    FILE *fp = fopen("demo.txt", "r");

    if (fp == NULL) {
        perror("demo.txt");
        return 1;
    }

    char buffer[100];
    if (fgets(buffer, sizeof(buffer), fp) == NULL) {
        if (ferror(fp) != 0) {
            perror("buffer");
            fclose(fp);
            return 1;
        }
    }

    clearerr(fp);
    puts(feof(fp) ? );

}


```

- 缓冲区错误：fprintf/fwrite 是先写到缓冲区，满了才写入磁盘
    - fflush(fp) 强制刷新缓冲区，将缓冲区数据立即写入磁盘，防止数据丢失
    - 正常程序退出return 0/exit(0) C标准库会自动调用fclose，而fclose内部会自动执行fflush

```c
fprintf(fp, "This is safe data protected by fflush");

fflush(fp);
_exit(1);

```

- 文件删除与重命名
    - 删除：int remove(const char *filename)
        - 成功返回0
        - 必须在文件关闭下删除
    - 重命名：int rename(const char *old_filename, const char *new_filename);
        - 成功返回0
        - 改名需要文件关闭，且新文件名不能与现有文件同名
        - 通过重命名来移动文件`rename("/tmp/demo.txt", "/home/user/demo.txt")`

- 问运算只能对整数型运算，负数以补码形式运算

- 库：预先编译好的代码集合（二进制文件）  .o 文件的集合
    - 静态库：链接阶段会被完整复制到最终可执行文件中
        - 后缀名Windows：.lib  (Library)
        - Linux : .a  (Archive)
    - 动态库：编译时不直接把、拷贝到可执行文件中，而是程序运行时由OS按需加载，可执行文件中仅保留对库的引用（符号表）
        - Windows：.dll (Dynamic Link Library)
        - Linux: .so (Shared Object)

- Linux下的静态库
    - 命名：`lib[your_lib_name].a`如`libmth.a`
    - 先将代码文件-c 生成目标文件，然后用ar工具将目标文件打包成.a静态库文件

## 优化
- 函数级优化
    - 调用级：减少函数调用，减少栈帧开销：内联函数incline、带参宏、尾递归优化
    - 数据级：减少拷贝：传指针、使用restrict消除别名
    - 编译器：static 限制作用域或者const关键词修饰

```c
// 内联函数：函数题很小而且频繁调用
static incline int max(int a, int b) {
    return (a > b) ? a : b;
}

// 尾递归：函数最后是调用自身
int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}

int factorial_tail(int n, int accumulator) {
    if (n == 0) return accumulator;
    return factorial_tail(n - 1, n * accumulator);
}

// 消除指针别名
// 明确告诉编译器a，b，result指向的内存不会重叠
void add_array(int *restrict a, int *restrict b, int *restrict result, int n) {
    for (int i = 0; i < n; i ++) 
        result[i] = a[i] + b[i];
}

```
